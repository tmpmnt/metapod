# RHEL 9 CIS Benchmarks with Ansible

## Description

This project is a set of Ansible playbooks that implement the CIS Benchmarks for RHEL 9. It is designed to automate the process of securing RHEL 9 systems based on the CIS recommendations. 

## Current Status

**WARNING**: This project is currently a work-in-progress and has not been thoroughly reviewed. It is **NOT RECOMMENDED FOR USE IN PRODUCTION ENVIRONMENTS** at this time. The playbooks are subject to change, and further testing and validation are required.

## Features

- Secures RHEL 9 systems following CIS Benchmarks.
- Divided into multiple roles for easier modular deployment.
